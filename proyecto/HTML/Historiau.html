Historias de Usuario

Sistema de Análisis y Monitoreo Canino

1.1. Definición de historia de usuario

Las historias de usuario sirven para describir funciones que el usuario requiere o desea poner en su sistema de manera clara para el usuario como el desarrollador usando un formato como: "me gustaría o quiero como (el tipo de usuario, ya sea administrador, usuario, etc) tener (funcionalidad deseada por el usuario) para (la intención de la funcionalidad deseada)".
