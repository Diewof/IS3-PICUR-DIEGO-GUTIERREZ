OBJETIVO GENERALE

Desarrollar un collar y un app para celular la cual de manera sinérgica permita, a través de sensores de movimiento, pulsaciones y sonido, monitorear las emociones del perro de una manera no invasiva, identificando y alertando de problemas de conducta y ofreciendo soluciones etológicas. mediante el uso de un collar fabricado de materiales seguros y cómodos, transmisión de datos por API Bluetooth,, algoritmos de análisis de comportamiento, una interfaz de aplicación intuitiva, y un sistema de sugerencias etológicas que fortalezca la relación entre perro y humano.

OBJETIVO EPECIFICO

Levantar los requerimientos a través de la interacción con los stakeholders o usuarios finales mediante encuestas y entrevistas

Crear un documento que especifique los requerimientos detallando todos los requerimientos funcionales y no funcionales.

Aplicar la metodología Scrum al desarrollo del proyecto, planteando un listado priorizado de los objetivos y determinando la duración del sprint incorporando las funcionalidades correspondientes a través del lenguaje de programación python para la implementación de la red neuronal, android studio para la app y arduino para el collar.

Desarrollar de manera iterativa las funcionalidades definidas para cada sprint planteado en la lista, asegurando que cada funcionalidad se pruebe y verifique

Redactar la documentación del código de una manera legible y comprensible

Desarrollar un plan de pruebas que incluya pruebas unitarias, de integración y de aceptación para verificar que cada requerimiento se cumple.
